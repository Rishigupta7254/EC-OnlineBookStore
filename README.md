# EC-OnlineBookStore
📚 EC-OnlineBookStore

An e-commerce based Online Book Store REST API application developed using Java, Spring Boot, Spring MVC, REST API, Hibernate/JPA, MySQL, Maven, and Exception Handling. This project allows users to manage books, carts, and orders through RESTful web services with proper exception handling and clean architecture.

🚀 Features
User Registration & Login API
CRUD Operations for Books
Search Books by Title or Author
Add Books to Cart
Place Orders
Admin Book Management
RESTful API Architecture
Global Exception Handling
Validation Handling
MySQL Database Integration
JSON Request & Response
