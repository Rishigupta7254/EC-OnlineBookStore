# 📚 EC-OnlineBookStore

EC-OnlineBookStore is a Full Stack Online Book Store application developed using Spring Boot, Spring Security, Hibernate, MySQL, and REST APIs.  
This project provides user registration, login authentication, file upload functionality, and book store management features.

---

# 🚀 Features

- User Registration
- User Login Authentication
- BCrypt Password Encryption
- REST APIs
- File Upload Functionality
- Spring Cache Support
- MySQL Database Integration
- Exception Handling
- Layered Architecture
- Logging using SLF4J
- Hibernate/JPA Integration
